Here we have a form section for a first name. You can use different col span sizes to fit the page how you want. We put each part of the for in it's own row. We also use the class "form-group" before the row class to say this is a part of the form. We put a label for class on the words First name, so when clicked it directs the user to the section to be filled out. Then in a separate div, but spanning the rest of the row, we put in an input for the user to fill out. Here we use the type of "text" because it will be text inputted there by the user. We give it an id to tag iit to the label, a name, a class of "form-control", which is bootstrap layout specifics, and a placeholder of "First Name", which the user sees before they start typing.
